# Pc Repairment Service

Python Fastapi app

## Running project

1. To run project firstly create python virtual enviroment in root project directory with this command:

```bash
python -m venv venv
```

2. After that enable venv and install requirements:

```bash
pip install -r requirements.txt
```

3. Run the project with this command:

```bash
python app/main.py
```

4. Open [link](http://localhost:8080) to access the app or open [documentation link](http://localhost:8080/docs#/) to access entrypoints.
